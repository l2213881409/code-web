互感器二次压降检测仪
检定系统



       互感器二次压降检测仪检定系统是基于现代数字化发展
趋势而设计出来了的更加方便、高效、便捷、准确的检定系
统，二次压降检测仪的检定也得到了质的飞跃。

       我们只需要在用户端输入数据，即可开始实验。实验结
束后也可以直接在客户端获取实验结果、打印实验报告。


开始使用

实验数据准备

输入设备信息

产品编号、产品名称、制造商、生产日期、送检日期、设备产地
输入实验数据

项目、数据上限、数据下限、档位、百分比、输入数据编号

通过计算机，进行简单操作，即可进行实验


用户只需要在电脑客户端进行启动实验，
计算机就会控制各项测试实验，
同时可以实时检测实验的进行效果

结果获取

实验结果拍照识别，存入计算机，为实时分析助力

拍  照



通过拍照获取显示屏结果

识  别



计算机识别照片，
读取出实验结果数据

存  储



结果存储于计算机，
方便数据分析调用

实验报告打印


输入设备标号
通过设备编号查找需要的二次压降检测仪数据

选择打印规格
可以选择需要打印的规格、打印的项目

打印报告
生成报告、可以直接打印
